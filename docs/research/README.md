# research/

Исследования, которые влияют на продуктовые и технические решения.

**Когда читать:** при проектировании новой фичи или выборе подхода — чтобы не переизобретать и опираться на факты.

## Что сюда идёт

- Нутрициология: формулы расчёта КБЖУ, клинические обоснования (BMR, TDEE, макронутриенты)
- Конкурентный анализ: что делают другие приложения и чем мы отличаемся
- UX-исследования: интервью, наблюдения, паттерны поведения пользователей
- AI/ML: подходы к персонализации, планированию меню, адаптации
