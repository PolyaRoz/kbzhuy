# Бобовые, субпродукты, паштеты и риеты

Эта группа продуктов хорошо работает в заготовках: дает белок, насыщенность, намазки, начинки и быстрые сборки.

## Бобовые

Бобовые можно использовать как:

- белковую основу;
- гарнир;
- загуститель супа;
- базу для намазки;
- начинку для овощей;
- добавку в салат;
- основу котлет.

Практические правила:

- Сухие бобовые лучше замачивать, если это требуется конкретному виду.
- Варите без кислоты до мягкости: кислота может замедлить размягчение.
- Солите ближе к концу или умеренно в процессе, ориентируясь на вид.
- Готовые бобовые удобно замораживать порциями.
- Консервированные бобовые - нормальный быстрый продукт, если промыть от лишней жидкости.

## Быстрые идеи с бобовыми

- Фасоль + томаты + специи = быстрый густой соус или рагу.
- Нут + тахини + лимон + чеснок = хумус.
- Чечевица + зажарка + бульон = суп.
- Фасоль + зелень + масло + кислота = салат.
- Бобовые + крупа + овощи = сытная тарелка без мяса.

## Субпродукты

Субпродукты требуют правильной подготовки, но дают сильный вкус и хорошую цену.

| Продукт | Как использовать |
| --- | --- |
| Печень | паштет, быстрое жаркое, соус, начинка |
| Сердца | тушение, шашлычки, рагу |
| Желудки | долгое тушение, суп, рагу |
| Язык | отварить, очистить, нарезать в салаты и закуски |
| Щечки | долгое тушение, насыщенный соус |

## Печень

- Не пересушивайте: печень быстро становится жесткой и мучнистой.
- Для паштета хорошо работают лук, морковь, сливочное масло, сливки, тимьян.
- После пробивания паштету нужно время на стабилизацию в холодильнике.
- Паштет удобен для завтраков, брускетт, перекусов и праздничной подачи.

## Паштеты, риеты, террины

Общая логика:

- паштет - гладкая пробитая масса;
- риет - волокнистая намазка из долго приготовленного мяса, птицы или рыбы;
- террин - плотная форма, которую нарезают ломтиками;
- парфе - особенно нежный паштет с большим количеством жира и аккуратной текстурой.

## Как включить в меню

- Паштет + хлеб + маринованный овощ = быстрый завтрак или перекус.
- Риет + салат + крупа = полноценный прием пищи.
- Язык + картофель + зелень + горчица = сытная тарелка.
- Чечевичные котлеты + йогуртовый соус + овощи = белковое блюдо без мяса.
- Хумус + овощи + яйцо + хлеб = быстрый обед.
