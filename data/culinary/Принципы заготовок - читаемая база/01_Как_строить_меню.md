# Как строить меню

Меню - это не расписание, которое нельзя нарушать. Это карта заготовок и вариантов сборки. Хорошее меню снижает хаос, но оставляет свободу.

## Этап 0. Желания и ограничения

Перед покупкой ответьте на вопросы:

- что давно хочется съесть;
- какие продукты уже есть дома;
- сколько раз в неделю реально готовить;
- сколько завтраков, обедов и ужинов нужно закрыть;
- будут ли гости, дорога, тренировки, поздние возвращения;
- какие продукты нужно срочно использовать;
- что можно купить замороженным без потери качества.

## Этап 1. Скелет недели

Соберите не блюда, а слоты.

| Слот | Примеры |
| --- | --- |
| Завтраки | сырники, вафли, яйца, каша, творог, брускетты |
| Суп | куриная лапша, овощной суп-пюре, томатный, грибной |
| Белок 1 | курица, индейка, говядина, рыба, фарш |
| Белок 2 | яйца, творог, бобовые, субпродукты, морепродукты |
| Гарнир | рис, булгур, гречка, паста, картофель, полента |
| Овощи | свежий салат, запеченные овощи, бланшированные овощи |
| Соусы | песто, томатный, йогуртовый, сливочный, азиатский |

## Этап 2. Матрица блюд

Один набор заготовок должен давать разные блюда.

Пример: куриный бульон, запеченное куриное филе, булгур, запеченные овощи, творожный сыр, зелень, томатная паста, песто.

Из этого можно собрать:

- горячую тарелку: булгур, курица, овощи, песто;
- салат: курица, булгур, зелень, моцарелла, кислая заправка;
- суп-пюре: запеченные овощи, бульон, томатная паста, сухарики, курица;
- брускетты: хлеб, творожный сыр, курица, вяленые томаты, зелень;
- завтрак: булгур с цукини, сыром и бульоном.

## Этап 3. Список покупок

Пишите список по зонам магазина:

- мясо, птица, рыба;
- молочные продукты и яйца;
- овощи, фрукты, зелень;
- крупы, паста, хлеб;
- консервы, соусы, специи;
- заморозка;
- расходники: пакеты, контейнеры, пергамент, фольга.

## Этап 4. План заготовок

После покупки разложите продукты по действиям.

| Действие | Что подходит |
| --- | --- |
| Разделать | курица, рыба, крупные куски мяса |
| Замариновать | птица, мясо, рыба, креветки |
| Сварить | бульон, крупы, яйца, бобовые |
| Запечь | овощи, корнеплоды, мясо, птица |
| Обжарить | грибы, зажарка, печень, фарш |
| Заморозить | бульон, котлеты, сырники, соусы, тесто |
| Оставить свежим | салаты, огурцы, зелень, часть фруктов |

## Этап 5. Оценка недели

После недели сохраните удачное:

- какие блюда реально съели;
- что осталось;
- какие заготовки ускорили жизнь;
- что надоело;
- чего не хватило: свежести, белка, овощей, соусов, сладкого;
- какие связки можно повторить.

После 6-10 таких циклов появится собственная библиотека меню, и планирование станет быстрым.
