# Конструктор блюд и замены

Интуитивная готовка начинается не с фантазии, а с понимания функций продукта.

## Формула блюда

```text
главный герой + фон + детали + текстура + финальная настройка
```

Пример:

- главный герой: курица;
- фон: булгур;
- детали: песто, сыр, зелень;
- текстура: сухарики или свежий огурец;
- настройка: соль, лимон, масло.

## Как менять блюдо

- Способ: жареные котлеты -> запеченные котлеты -> фрикадельки в соусе.
- Форма: большое изделие -> маленькие порции -> начинка -> салат.
- Главный продукт: говяжий фарш -> куриный -> индейка -> бобовые.
- Гарнир: картофельное пюре -> гратен -> драники -> запеченный картофель.
- Соус: майонез -> сметана с горчицей -> йогурт с зеленью -> томатный соус.
- Кислота: клюква -> брусника -> смородина -> вишня -> лимон.

## Правило 3-4 ингредиентов

Если придумываете блюдо с нуля, начните с 3-4 главных компонентов. Потом пробуйте и добавляйте:

- соль, если вкус пустой;
- кислоту, если тяжело;
- жир, если сухо;
- сладость, если слишком кисло или остро;
- умами, если вкус плоский;
- хруст, если все мягкое.

## Синонимы продуктов

| Нужно | Можно заменить |
| --- | --- |
| Курица | индейка, утка, креветки, говядина, рыба |
| Говядина | индейка, курица, баранина, грибы с бобовыми |
| Форель | горбуша, семга, кета, скумбрия |
| Пармезан | другой твердый сыр, выдержанный сыр |
| Моцарелла | сулугуни, адыгейский сыр, фета |
| Кабачок | цукини, тыква, морковь, капуста |
| Булгур | рис, кускус, перловка, киноа, гречка |
| Сливки | сметана, йогурт, творожный сыр, кокосовое молоко |
| Песто | зелень с маслом, горошкомоле, ореховый соус |

## Быстрые сборки

- Салат: белок + зелень + крупа + кислая заправка + сыр.
- Боул: крупа + белок + овощи + соус + семечки.
- Суп: бульон + овощи + белок + крупа/лапша + зелень.
- Брускетта: хлеб + намазка + белок/овощи + кислота + зелень.
- Запеканка: готовая база + белок + соус + сыр + духовка.
