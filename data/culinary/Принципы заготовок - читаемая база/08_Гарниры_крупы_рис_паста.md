# Гарниры, крупы, рис и паста

Гарнир - это не второстепенная часть. Он связывает белок, соус и овощи в нормальное блюдо.

## Общие правила круп

- Большинство круп лучше промывать.
- Для рассыпчатого гарнира важно не переварить и дать крупе отдохнуть под крышкой.
- Для каши нужно больше жидкости и более мягкая текстура.
- Крупы долгой варки удобно готовить заранее и хранить порциями.
- Готовую крупу можно оживить бульоном, маслом, соусом, сыром, зеленью.

## Быстрые пары

| Крупа | С чем особенно хороша |
| --- | --- |
| Рис | рыба, курица, овощи, азиатские соусы |
| Булгур | курица, зелень, томаты, сыр, запеченные овощи |
| Перловка | грибы, тыква, сыр, мясной бульон |
| Гречка | грибы, сливочное масло, яйца, говядина |
| Пшено | тыква, овощные котлеты, молочные каши |
| Кускус | зелень, лимон, овощи, рыба |
| Полента | сыр, грибы, томатный соус, мясо |

## Рис

- Для рассыпчатого риса промывайте до прозрачной воды.
- Для ризотто рис не промывают: крахмал нужен для кремовости.
- Рис хорошо работает как база для жареного риса: лучше использовать вчерашний охлажденный рис.
- Рис можно использовать в супах, фаршированных овощах, запеканках.

## Паста

- Вода должна быть хорошо посолена.
- Варите до `al dente`, особенно если паста потом идет в соус или запеканку.
- Оставляйте немного воды от варки для эмульсии соуса.
- Пасту лучше соединять с соусом на сковороде, а не просто поливать сверху.

## Заготовки

- Сварить крупу на 2-3 дня.
- Заморозить часть гарнира долгой варки.
- Держать бульон для разогрева круп.
- Делать салаты с крупами: крупа + белок + овощи + кислая заправка.
- Использовать остатки гарнира в котлетах, запеканках, супах.
